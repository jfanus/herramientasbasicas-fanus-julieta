#  Análisis de ventas – Industria cosmética (2025)

**Materia:** Herramientas básicas para el Análisis de Datos  
##  Objetivo del proyecto

El objetivo de este trabajo es analizar el comportamiento de las ventas dentro de la industria cosmética durante el año 2025, utilizando herramientas de análisis de datos para identificar tendencias, patrones de consumo y oportunidades comerciales.
##  Dataset

Se trabajó con un conjunto de datos de ventas de productos de maquillaje que incluye las siguientes variables:

- Fecha  
- Marca  
- Tipo de producto  
- Canal de venta  
- Unidades vendidas  
- Precio e ingresos en USD
- ##  Limpieza y preparación de datos

El procesamiento de los datos se realizó en Python utilizando Google Colab.

Las principales tareas fueron:

- Revisión de tipos de datos  
- Tratamiento de valores nulos  
- Eliminación de duplicados  
- Estandarización de variables
- ##  Dashboard en Power BI

El dashboard incluye:

###  KPIs
- Ventas totales (USD)  
- Unidades vendidas  
- Precio promedio  

###  Visualizaciones
- Evolución mensual de ventas  
- Ventas por marca  
- Ventas por canal  
- Top 5 productos (unidades)  
## Aspectos destacados

- El canal online presenta el mayor volumen de ventas.  
- Se observan variaciones mensuales en el comportamiento de las ventas.  
- Algunas marcas concentran la mayor parte de los ingresos.  
- Los productos más vendidos reflejan tendencias claras de consumo.
## Herramientas utilizadas

- Python (Pandas)  
- Google Colab  
- Power BI  
- GitHub
  ## Conclusión

A partir del análisis realizado, se logró obtener una visión integral del desempeño de ventas en la industria cosmética durante el año 2025. La combinación de herramientas de procesamiento de datos y visualización permitió identificar patrones relevantes en el comportamiento del mercado.

En primer lugar, se destaca el predominio del canal online como principal medio de venta, lo que evidencia una clara tendencia hacia la digitalización del consumo. Asimismo, el análisis temporal mostró variaciones a lo largo del año, lo que podría estar vinculado a factores estacionales o a estrategias comerciales específicas.

Por otro lado, se observó una concentración significativa de los ingresos en determinadas marcas, lo que sugiere un posicionamiento diferencial dentro del mercado. En cuanto a los productos, las categorías más vendidas reflejan preferencias definidas por parte de los consumidores, lo que resulta clave para la toma de decisiones comerciales.

En conjunto, los resultados obtenidos permiten no solo describir el comportamiento de las ventas, sino también generar información valiosa para orientar estrategias futuras, optimizar la oferta de productos y mejorar la eficiencia de los canales de comercialización.
